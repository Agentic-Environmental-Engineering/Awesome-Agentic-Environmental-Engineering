# Contributing

Thank you for helping improve this survey repository. We welcome additions of papers, benchmarks, environments, toolkits, and corrections.

## What to Add

Good additions should be directly related to agentic environments for large language model agents, including:

- Environment benchmarks for GUI, web, tools, code, games, embodied tasks, science, finance, medicine, or cross-domain agents.
- Environment synthesis methods, including symbolic, neural, simulator-based, or world-model-based construction.
- Environment evaluation methods for correctness, diversity, complexity, fidelity, or reliability.
- Agent-environment evolution methods, including curriculum learning, self-play, online RL, trajectory synthesis, and environment scaling.

## Entry Format

Please use this format in both `README.md` and `resources/papers.md`:

```markdown
1. **Title**, *Authors*, Venue Year. [Paper](https://example.com/paper)
```

If there is no stable paper or project URL yet, use plain text for the title and add the link later.

## Pull Request Checklist

- The work is placed under the most relevant taxonomy section.
- The entry is added to both the main `README.md` and `resources/papers.md` under the same category.
- The link is stable and publicly accessible.
- The description is short, factual, and not promotional.
- Duplicate entries are avoided.
- If a new category is needed, explain why in the pull request.

## Corrections

If you find an inaccurate classification, broken link, missing citation, or unclear description, please open an issue or pull request with the correction.
